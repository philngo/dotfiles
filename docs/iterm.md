# iTerm2 Setup

Note: Opening iTerm2 triggers installation of Xcode Command Line Tools.

## Settings
1. Appearance
    1. Dimming
        1. Dimming amount: 15
        2. Dimming affects only text, not background: On
    2. Tabs
        1. Show tab bar even when there is only one tab: On
        2. Do not stretch tabs to fill bar: On
2. Profiles
    1. General
        1. Working Directory: ~/dev
    2. Colors
        1. Color Presets: Solarized Dark
        2. Minimum Contrast: 33 (prevents invisible text in some CLI tools)
    3. Terminal
        1. Notifications
            1. Silence Bell
        2. Scrollback lines: Unlimited
    4. Window Arrangements
        1. Set up preferred arrangements for different workflows
